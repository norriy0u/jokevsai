# 🎤 Joke vs. AI Joke

A high-energy, comedy club-themed web game where players test their "Joke IQ" by guessing whether a joke was written by a classic stand-up comedian or generated by AI. 

## ✨ Features
- **Offline Playable**: Comes pre-loaded with a hardcoded bank of classic human jokes and subtle AI-generated jokes. No API keys required!
- **Procedural Jazz Club Audio**: Custom Web Audio API soundscape featuring a continuous sine-wave jazz bassline, brush cymbals, rim-shots, and sad trombones. 
- **Theatrical UI**: CSS curtain animations, dynamic spot lights (radial gradients), and a custom SVG hourglass timer.
- **Canvas Certificate**: Generates a downloadable "Comedian's Certificate" image displaying your final Joke IQ score and classification.
- **Persistent Leaderboard**: Uses localStorage to track your high scores and analyze which jokes fool you the most.

## 🚀 Getting Started
Simply open `index.html` in any modern web browser or serve it locally:
```bash
python -m http.server 8080
```

## 🛠️ Tech Stack
- **HTML5 & CSS3**: Glassmorphism, CSS shapes, clip-paths.
- **Vanilla JavaScript**: Logic, state management, and Canvas export.
- **Web Audio API**: Real-time sound generation without external audio files.
